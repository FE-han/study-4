## 문제 설명

N 개의 데이터가 주어졌을 때, 체크 암호를 만드는 프로그램을 작성하세요.  <br>


## 자료 구조
-n  <br>
타입 : int <br>
n개의 데이터를 입력받을 변수

-count  <br>
타입 : int <br>
모든 데이터를 더한 값을 나타내는 변수


## 풀이 과정
예시 :
```txt
1. N개 데이터 입력받음
2. 입력받은 5개 데이터를 반복문으로 돌면서 모두 더하기
3. 문자열로 형변환후 10번째까지 출력

```

## 코드 구현
```python

n = int(input())           

count = 0                       

for i in range(n):
    count += int(input())     
    
print(str(count)[:10]) 

```
