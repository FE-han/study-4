
## 도어락 비밀번호

### 문제 설명

- 태어난 연도를 역순으로 바꿔서 합한 수를 출력하는 프로그램<br>


### 자료 구조

- a, b<br>
    - 타입 : 숫자
    - 저장 데이터 : 태어난 연도를 역순으로 변경해서 숫자로 저장

### 풀이 과정

```txt
1. a, b를 역순으로 변경해서 숫자로 저장

2. a + b 출력

```

### 코드 구현
사용 언어 : 파이썬<br>

 
```python
# 풀이과정 1
a, b = list(map(lambda x : int(x[::-1]), input().split()))

# 풀이과정 2
print(a+b)

```
