
## 도도새의 업무일지

### 문제 설명

- 주어진 업무일지 B를 보고 업무일지 A를 구하는 프로그램<br>


### 자료 구조

- day<br>
    - 타입 : 숫자
    - 저장 데이터 : 업무일지 길이

- B<br>
    - 타입 : 리스트
    - 저장 데이터 : 업무일지 B 내용

- A<br>
    - 타입 : 리스트
    - 저장 데이터 : 업무일지 A가 저장될 내용

### 풀이 과정

```txt
1. day와 B에 입력 받는다

2. A에 빈 리스트 선언

3. B의 길이 만큼 반복하며 B에서 A를 계산해서 넣는다

4. A를 문자열로 변환해서 출력

```

### 코드 구현
사용 언어 : 파이썬<br>

 
```python
# 풀이과정 1
day, B = [list(map(int, input().split())) for _ in range(2)]

# 풀이과정 2
A = []

# 풀이과정 3
for i in range(len(B)):
    A += [B[i] * (i+1) - sum(A)]

# 풀이과정 4
print(' '.join(list(map(str, A))))

```
